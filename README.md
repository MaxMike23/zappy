# Zappy (working title)
### An App designed to help streamline AV and IT tasks for AV Field Technicians, Programmers, and Engineers
Zappy is an application designed to assist AV/IT technicians, programmers, and engineers with designing, installing, programming, testing, and troubleshooting networked AV or any other IP devices on-site. The goal is to streamline AV / IT tasks for both new and experienced technicians while providing an opportunity to learn key concepts related to IT, AV, and Security networking and programming.
